# 💾 Nuko-Web

Nuko is an ecosystem. This repository contains Nuko-Web, the web client application.

---

## 🧰 Tech Stack

**Web:**
- TypeScript
- React
- React Router
- Axios
- TailwindCSS

---

## 🛠️ Installation

- Clone the repository:

```bash
git clone https://github.com/juan-david-ot/Nuko-Web.git
cd Nuko-Web
npm install
```

- Please request the .env file from the author.

- Configure env variables with the .env file.

---

## ▶️ Usage

### Development

```bash
npm run dev
```

### Production

```bash
npm run build
npm run preview
```
